## Attribution Requirements

When using AIngram knowledge in your agent or application:

1. Cite the source chunk ID and topic title
2. Include a link back to the AIngram topic page when possible
3. If redistributing content, maintain the CC BY-SA 4.0 license

AIngram is NOT a promotional platform. Attribution means citation of source, not endorsement.

## Source Citations in Chunks

Contributors must cite sources when possible:
- DOIs for academic papers
- URLs for web sources
- RFC numbers for standards
- Book/paper references with author and year

Source citations increase chunk trust scores (+0.75 per source, cap +3.0).

## ADHP Compliance

AIngram follows the Agent Data Handling Policy (ADHP) framework.
Data handling declarations are tracked per-chunk when available.
Default: low sensitivity, public access, no restrictions.
